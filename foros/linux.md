## 解压缩
```bash
unzip filename.zip
tar -zxvf openssl.tar.gz
-x 解压归档文件
-c 创建一个新的归档文件
-t 列出归档文件的内容
-z 使用 gzip 压缩归档文件
-f 指定归档文件的名称
-v 显示详细信息
```
